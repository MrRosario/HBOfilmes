## Configuração

### Pré-requisitos

- Instalar [Node.js] que inclui [Node Package Manager][npm]

### Configurando o Projeto

clonar o projeto:

```
git clone https://github.com/MrRosario/HBOfilmes.git
```

Rodar a aplicação:

```
cd [NOME PROJETO]
npm start
```

[node.js]: https://nodejs.org/
[npm]: https://www.npmjs.com/get-npm
